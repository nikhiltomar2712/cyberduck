# Cybersecurity Fundamentals

## Core security goals

- **Confidentiality:** keep sensitive data away from unauthorized people.
- **Integrity:** ensure data and systems are accurate and unchanged unless authorized.
- **Availability:** keep services usable when people need them.

## Common terms

- **Asset:** something valuable, such as data, accounts, laptops, APIs, or servers.
- **Threat:** a possible cause of harm.
- **Vulnerability:** a weakness that can be exploited.
- **Risk:** the likelihood and impact of a threat exploiting a vulnerability.
- **Control:** a safeguard that reduces risk.

## Good security habits

- Use unique passwords stored in a password manager.
- Enable multi-factor authentication for important accounts.
- Keep systems updated.
- Back up important data and test restores.
- Apply least privilege: give users and services only the access they need.
