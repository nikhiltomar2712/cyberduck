# Cybersecurity Glossary

- **2FA/MFA:** extra login factor beyond a password.
- **CVE:** public identifier for a known vulnerability.
- **Encryption:** transforming data so only authorized parties can read it.
- **Firewall:** control that filters network traffic.
- **Hash:** one-way fingerprint of data.
- **Least privilege:** giving only required permissions.
- **Patch:** update that fixes bugs or vulnerabilities.
- **Phishing:** tricking people into revealing information or taking unsafe actions.
- **Zero day:** vulnerability unknown to the vendor or without a patch.
