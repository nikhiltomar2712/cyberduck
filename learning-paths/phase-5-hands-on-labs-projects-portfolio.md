# Phase 5: Hands-on Labs, Projects & Portfolio

> **Ethics & Authorization**: All labs must be local or explicitly authorized.

## Learning objectives
- Turn skills into documented outcomes.
- Build a portfolio that demonstrates evidence, risk reasoning, and mitigations.
- Automate small defensive tasks.

## Weekly plan (template)

### Required deliverables
- 2 capstone write-ups (full reports)
- 6–10 shorter portfolio write-ups
- 1–2 small automation scripts or tools

### Capstone ideas
- Blue-team home lab capstone
- SOC sprint project: triage runbook + detection plan

### Portfolio write-up format (must use templates)
- Summary
- Scope & authorization
- Evidence collected
- Findings
- Impact
- Mitigations
- Verification steps
- What you would do next

## Success checkpoints
- Portfolio includes reproducible documentation.
- You demonstrate defensive reasoning, not exploitation steps.
