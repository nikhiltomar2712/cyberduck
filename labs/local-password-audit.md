# Lab: Local Password Audit

Goal: learn how to evaluate your own password habits without exposing secrets.

## Rules

- Use only your own accounts and devices.
- Do not store real passwords in this repo.
- Do not share password dumps.

## Exercise

1. List important accounts: email, GitHub, cloud, bank, school, work.
2. Check whether each account has a unique password.
3. Check whether MFA is enabled.
4. Move weak or reused passwords into a password manager and rotate them.
5. Save only a summary, never the passwords themselves.

## Reflection

- Which account would hurt most if compromised?
- Which account recovery method is weakest?
- What can be improved today?
